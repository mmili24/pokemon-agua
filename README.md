# 🌊 Pokédex - Nucli Central HTML5

Maquetació del nucli central d'informació d'una Pokédex amb HTML5 semàntic. 
Projecte per aprendre a estructurar contingut amb `<main>`, `<section>`, `<article>` i `<aside>`.

## 🚀 Com executar-ho

1. Clona el projecte.
2. Obre `index.html` al navegador.

## 🛠️ Tecnologies

* HTML5
* Git & GitHub
